## About this project

This project is a simplified cloud storage application inspired by services like Google Drive. It enables authenticated users to upload, organize, and manage files within folders. 

Built with Express and Prisma, it implements session-based authentication using Passport.js with persistent session storage in the database. Users can create, update, and delete folders, and upload files into specific directories using Multer for file handling. 

The project integrates Cloudinary for cloud-based file storage, with file URLs stored in the database to support scalable and reliable access.

It is part of the [Odin Project](https://www.theodinproject.com) curriculum.
