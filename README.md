#File Uploader

##Description
File Uploader is a web application made to interpret the functionality of Google Drive. The platform enables users to upload, store, and organize files in a folder structure. Designed as a backend-focused project, it serves as a exercise in building scalable server-side applications using Express.js and Prisma ORM.

This project reflects an ongoing effort to deepen proficiency in full-stack web development through hands-on experimentation and problem-solving.
